# 더 신라 테이블 (THE SILLA TABLE)

> 경주·신라 헤리티지를 담은 **MICE 케이터링·단체도시락 브랜드 웹사이트** 프로젝트

학회·기업·관공서 대상 **대규모 행사 케이터링**, **단체도시락**, **헤리티지 다이닝**, **쿠킹클래스**를 아우르는 브랜드 웹사이트입니다.
모나테이블([monatable.com](https://monatable.com)) 구조를 롤모델로 하되, **B2B 대규모 실행력 + 경주·신라 헤리티지 + 인바운드 특수식이(할랄·비건 등) 대응**을 차별점으로 재구성했습니다.

## 🖥️ 미리보기 여는 법

별도의 설치나 서버 없이 바로 확인할 수 있습니다.

1. `홈페이지_미리보기/index.html` 파일을 **더블클릭**합니다.
2. 기본 브라우저(크롬·엣지·사파리 등)에서 홈페이지 디자인이 열립니다.

## 🧩 현재 상태 및 한계

## 🚀 향후 로드맵 (개발계획서 기준)

1. **개발 환경 구축** — Node.js 설치
