## 0. 왜 블로그를 리뉴얼하는가

- **황오부엌**은 동네 밥집 감성으로, 대규모 B2B 케이터링 브랜드 이미지와 맞지 않는다.
- 블로그는 **네이버 검색 상위 노출의 핵심 자산**이다. 학회 사무국·기업 총무가 "경주 학회 케이터링", "경주 단체도시락"을 검색할 때 블로그 후기가 걸려야 문의로 이어진다.
- 홈페이지·인스타·플레이스와 **하나의 브랜드(더 신라 테이블)**로 통일해 신뢰를 준다.

> **블로그의 역할 정의**: SEO 콘텐츠 허브 → 검색 유입을 만들고, 모든 글의 종착지를 **홈페이지 견적 문의폼**으로 연결한다.

## 2. 블로그 디자인(스킨) 리뉴얼

홈페이지와 같은 **크림·테라코타 프리미엄 톤**으로 통일한다.

| 요소 | 리뉴얼 방향 |
|---|---|
| **타이틀 이미지** | 로고 + 슬로건 "천년 고도, 감동을 한 상에" + 대표 음식 사진. 홈페이지 무드와 동일 |
| **프로필/대표 이미지** | 로고 통일 |
| **투명 위젯(사이드바)** | ① 홈페이지 바로가기 ② 전화(010-2388-1457) ③ 카카오톡 채널 ④ 인스타그램 ⑤ 견적문의 버튼 |
| **상단 메뉴(프롤로그)** | 대표 카테고리(실적·도시락·메뉴·이용안내)를 상단 노출 |
| **모바일 커버** | 모바일 유입이 많으므로 모바일 커버 이미지·소개도 통일 |

> 실행: 네이버 블로그 → 관리 → 꾸미기 설정 → 스킨/레이아웃·위젯. 타이틀은 이미지 제작 필요(캔바 등).

## 3. 카테고리 구조(IA)

기존 잡다한 카테고리를 정리하고, **검색 키워드 + 서비스 구조**로 재편한다.

```
더 신라 테이블 블로그
├── 📢 공지 / 브랜드 소개
│     └── (황오부엌 → 더 신라 테이블 리뉴얼 안내 고정글)
├── 🍽️ 케이터링 실적 (핵심)
│     ├── 학회·학술대회 케이터링
│     ├── 기업 행사 케이터링
│     └── 관공서·공공행사 케이터링
├── 🎁 단체도시락 / 프리미엄 도시락
├── 🥂 뷔페·디저트·핑거푸드
├── 🍳 메뉴 이야기 / 음식 비하인드
├── 🏛️ 브랜드 스토리 (경주·신라 헤리티지)
├── 📋 견적·이용 안내 (자주 묻는 질문)
└── 🌏 인바운드·특수식이(할랄·비건) 대응
```

> 원칙: 카테고리명에도 **검색 키워드**를 녹인다. (예: "학회 케이터링", "단체도시락")

## 4. 네이버 검색 최적화(SEO) 전략

네이버는 **C-Rank(블로그 신뢰도) + DIA(문서 품질)** 로직으로 노출 순위를 정한다. 핵심은 **"특정 주제를 꾸준히, 깊이 있게"** 쓰는 것.

### 4.1 키워드 전략
- **핵심 키워드**: 경주 케이터링 / 경주 학회 도시락 / 경주 단체도시락 / 대구·경북 MICE 케이터링 / 기업 행사 케이터링 / 관공서 단체도시락 / 할랄 케이터링 / 경주 출장뷔페
- **롱테일(전환율 높음)**: "경주 화백컨벤션센터 케이터링", "1000명 학회 도시락 업체", "경주 기업 행사 뷔페 추천"
- 도구: 네이버 **검색광고 키워드도구**, **블랙키위**로 검색량 확인 후 제목에 배치

### 4.2 제목 공식
`[핵심키워드] + [구체 정보/규모] + [후기/추천]`
- 예: "경주 학회 케이터링｜1,200명 학술대회 단체도시락 후기"
- 예: "경주 기업 행사 뷔페｜LG전자 리더십 행사 케이터링 실적"

### 4.3 본문 품질 기준 (DIA 대응)
- **글자수 1,500자 이상** (정보 충실)
- **사진 10~15장** (직접 촬영 원본, 순서·설명 포함)
- **동영상/릴스 1개** 삽입 시 체류시간↑ (노출 유리)
- 소제목·리스트로 **가독성** 확보
- 본문에 핵심 키워드 자연스럽게 3~5회 반복 (과도한 반복=저품질 판정 주의)
- 글 하단 **홈페이지 링크 + 전화 + 지도** 고정
- **태그 10개 내외** (키워드 기반)

### 4.4 하지 말아야 할 것 (저품질 방지)
- 타 블로그 글·사진 복붙 ❌ / 과장·허위 ❌ / 키워드 남발 ❌ / 유사 글 반복 ❌

## 5. 콘텐츠 전략

### 5.1 콘텐츠 기둥 5가지 (돌려가며 발행)
| 기둥 | 목적 | 예시 글 |
|---|---|---|
| ① 행사 실적·후기 | 검색·신뢰 (최우선) | "○○학회 1,200명 단체도시락 후기" |
| ② 메뉴·음식 | 비주얼·식욕 자극 | "케이터링 대표 메뉴 구성 소개" |
| ③ 정보·가이드 | 검색 유입·전문성 | "학회 도시락 준비 체크리스트 5가지" |
| ④ 브랜드 스토리 | 차별화·팬 형성 | "경주 신라 헤리티지를 담은 이유" |
| ⑤ 이용 안내·FAQ | 전환 지원 | "케이터링 견적 문의 방법·최소 인원 안내" |

### 5.2 "행사 1건 = 콘텐츠 3채널" 원칙
행사 1회 → **블로그 후기(길게)** + 인스타 사진/릴스 + 플레이스 사진. 한 번의 행사로 3채널을 동시에 채운다.

### 5.3 발행 계획
- **주 1~2회** 꾸준히 (네이버는 꾸준함을 높이 평가)
- 초기 4주는 "리뉴얼 집중기" — 대표 실적글 위주로 8~10편 발행해 기반 구축
- 행사 있을 때마다 24~48시간 내 후기 발행(최신성)

```
5. 고객 반응 / 마무리 코멘트
```

```
1. 문제 제기 (담당자 고민)
2. 체크포인트 1~5 (인원·예산·위생·정시·특수식이)
3. 우리 브랜드가 해결하는 방식
4. [CTA] 견적 문의 안내
```

## 7. 리브랜딩 전환 실무 (황오부엌 → 더 신라 테이블)

1. **별명·블로그명·소개·프로필 이미지** 변경
2. **타이틀 스킨** 새 디자인 적용
3. **카테고리 재편** (위 IA대로) — 기존 글은 새 카테고리로 이동
4. **리뉴얼 공지글 작성 + 상단 고정**: "황오부엌이 '더 신라 테이블'로 새롭게 시작합니다" (기존 방문자 안내, 브랜드 스토리 포함)
5. **기존 글 유지** (삭제 금지 — 검색 자산). 대표 글은 제목·내용 보강(리라이팅)
6. **모든 글 하단 CTA 통일** (홈페이지 링크·전화·지도)

> 기존 글이 저품질이면 삭제보다 **비공개 전환 후 리라이팅 재발행**을 권장.

## 8. 채널 연계 (일관성)

- **블로그 → 홈페이지**: 모든 글 하단에 홈페이지 견적 문의 링크
- **홈페이지 → 블로그**: 홈페이지 "현장스케치(FIELD NOTES)"에서 블로그 글 연결
- **플레이스 → 블로그**: 플레이스 소식·사진과 블로그 후기 연동
- **인스타 → 블로그**: 인스타 프로필 링크/스토리로 블로그 후기 유도
- **네 채널 프로필·소개·로고·링크 100% 통일**
